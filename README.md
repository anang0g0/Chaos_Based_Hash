# Chaos_Based_Hash


原理：データバッファをコピーしてそれに置換と非線形変換（AESのS-box）したものをコピー前のバッファにXORしているつもり。

20200711

処理が遅かったので高速化しました。
ディスクアクセスが原因で大きなファイルの処理に時間が書かていましたが、
読み込みバッファのサイズを大きくしたことでより大きなサイズのファイルに対しても
高速にハッシュ値を計算できるようになりました。（chash.c）

細かい部分を直しました。
